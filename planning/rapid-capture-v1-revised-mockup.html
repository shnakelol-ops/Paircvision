V1 Revised — Change Summary
  • Setup screen added — sport selector, team name inputs, attack direction and half-duration pickers before match start.
  • Scoreboard displays totals in parentheses — e.g. "1-04 (07)" so net score is always visible alongside goals + points split.
  • HT chip on clock — a blue "HT" badge appears in the score-center block when the half ends, distinct from the running period label.
  • SHOT button uses armed state — highlighted orange (#f0883e) once tapped, signalling the user must next tap a pitch zone to place the attempt.
  • Signal / insight chip — amber left-bordered chip below the keyboard surfaces automated pattern alerts (e.g. turnovers clustering in a channel).
  • Half-time overlay on pitch — semi-transparent card with score summary and RESUME 2H / MATCH SUMMARY actions; keyboard and controls dimmed to 35% opacity.
  • FOR / OPP context colours — FOR active = #1f6feb blue, OPP active = #b91c1c red, visible in controls row and status dot.
  • Event dot palette on pitch — red (#ef4444) for OPP events, teal (#22d3ee) for contested / puckout, green (#4ade80) for FOR events.
